# Overview
**Floe** is an AI technical writing assistant. Floe can make it easier to write and maintain documentation at scale by:
- **Validating** documentation against rules
- **Suggesting** improvements to documentation 
- **Writing** documentation from code *(coming soon)*
- **Updating** documentation when code changes *(coming soon)*
- **Translating** documentation into other languages *(coming soon)*
