# Prerequisites
The following guide will walk you through setting up a local development
environment for Floe.

Before you begin, be sure to create accounts with the following services, and
install the following packages. All 3rd party services should have a free tier
available.

macOS, Windows, and Linux are supported.

### Accounts
- [GitHub](https://https://github.com/)
- [Vercel](https://vercel.com/)
- [PlanetScale](https://planetscale.com/)
- [SendGrid](https://sendgrid.com/)
- [OpenAI](https://openai.com/)
- [LangFuse](https://langfuse.com/) (Optional) 

### Packages
- [Node.js 18.19](https://nodejs.org/en) or later
- [pnpm 8.x](https://pnpm.io)
- [PlanetScale CLI](https://planetscale.com/features/cli)
